WebRTC (Web Real-Time Communication) is an open source project to enable realtime communication of audio, video and data in Web and native apps.
In January 2021, the WebRTC 1.0 specification transitioned from Candidate Recommendation to Recommendation.

WebRTC has several JavaScript APIs:

  • getUserMedia(): capture audio and video.
  • RTCPeerConnection: stream audio and video between users.
  • RTCDataChannel: stream data between users.
The WebRTC specification includes APIs for communicating with an ICE (Internet Connectivity Establishment) Server, but the signaling component is not part of it.

Signaling servers:

  • RTCMultiConnection.js
  • DataChanel.js
  • Socket.io over Node.js
  • WebSocket over Node.js
  • WebSync for Signaling - useful only for .NET developers
  • XHR/XMLHttpRequest Signaling r-stretch useful for both .NET and PHP developers!
  • RTCMultiConnection

WebRTC APIs use STUN servers to get the IP address of your computer, and TURN servers to function as relay servers in case peer-to-peer communication fails.
